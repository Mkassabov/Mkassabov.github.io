Mtss Utopia Project
A demo for a cloud based system to manage and support students. This is a demo, in an ideal scenario student info will remain in a database hosted by the school system. Only certain users can see certain information regarding the status of the students within the system. This is done via Google account integration.
Started: 2018-1-15
